\chapter*{Glossaire}
\begin{description}
\item[Définition 1 : Le Makefile]
est un fichier utilisé par la commande make des distributions GNU/Linux permettant l'automatisation de tâches simples telles que la compilation.
\item[Définition 2 : Une Distribution] 
de GNU/Linux est une version d'un système d'exploitation, il y en a plusieurs car il y a plusieurs systèmes d'exploitation utilisant le même noyau : Ubuntu, Debian, Red Hat.
\item[Définition 3 : Un Paquet] 
Debian est un paquet rassemblant les sources d'un programme et comment les installer, ainsi que les dépendances nécessaires. On les trouve typiquement dans les archives Debian et elle servent à installer des programmes facilement sous Ubuntu ou Debian.
\item[Définition 4 : Un Flux ou Flot de données] 
est la succession de données d'un certain type, elle peut se poursuivre pendant un temps indéfini.
\item[Définition 5 : Une Allocation] 
est l'opération qui consiste à réserver de la place dans la mémoire, c'est une opération qui demande un peu de ressources et qui est donc à utiliser le moins possible.
\item[Définition 6 : Le mode]
d'une série de nombres est le nombre qui apparait le plus de fois dans la série.
\end{description}
