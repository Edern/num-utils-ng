\chapter*{Glossaire}
\begin{description}

\item[Allocation] : l'alloction est l'opération qui consiste à réserver de la place dans la mémoire, c'est une opération qui demande un peu de ressources et qui est donc à utiliser le moins possible.
\item[autotools] : outils fournis par le projet GNU pour faciliter la fabrication de paquets à partir du code source d'un programme.
\item[Distribution] : Une distribution de GNU/Linux est une version d'un système d'exploitation, il y en a plusieurs car il y a plusieurs systèmes d'exploitation utilisant le même noyau : Ubuntu, Debian, Red Hat.
\item[Flux ou Flot de données] : le flot de données est la succession de données d'un certain type, qui peuvent se poursuivre pendant un temps indéfini.
\item [fuite de mémoire] : occupation croissante et non contrôlée ou non désirée de la mémoire d'un ordinateur. 
\item[Gestionnaire de version] : logiciel qui permet de stocker un ensemble de fichiers en conservant la chronologie de toutes les modifications qui ont été effectuées dessus.
\item[Langage compilé] : langage dans lequel le code est traduit en langage binaire pour une compréhension direct du processeur. Un programme écrit en langage compilé est traduit en instructions lisibles par la machine et peut être exécuté indépendamment de tout autre programme.
\item[Langage interprété] : langage dans lequel le code n'est pas compilé . Un programme écrit en langage interprété n'est pas exécuté directement par la machine mais par un autre programme.
\item[logiciel open source] : logiciel dont l'utilisation, l'étude, la modification et la duplication en vue de sa diffusion sont permises, techniquement et légalement.
\item[Makefile] : un Makefile est un fichier utilisé par la commande make des distributions GNU/Linux permettant l'automatisation de tâches simples telles que la compilation.
\item[mode] : le mode d'une série de nombres est le nombre qui apparait le plus de fois dans la série.
\item[Paquet] : un paquet Debian est un paquet rassemblant les sources d'un programme et comment les installer, ainsi que les dépendances nécessaires. On les trouve typiquement dans les archives Debian et elle servent à installer des programmes facilement sous Ubuntu ou Debian.

\end{description}
