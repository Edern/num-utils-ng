NAME

numround - A program that rounds off numbers it encounters.  

SYNOPSIS

numround [-cfh] <FILE>

| numround [-cfh] (Input on STDIN from pipeline.)

numround [-cdfhV] (Input on STDIN. Use Ctrl-D to stop.)  

DESCRIPTION

numround will simply round decimal numbers to the nearest integer or to a factor of any number using the -n option. You can also force it to round up (ceiling) or round down(floor) using the -c and -f options.  

OPTIONS

-c
Force the number to be rounded up. Ceiling.
-f
Force the number to be rounded down. Floor.
-n <n>
Round numbers to the nearest factor of <n>.
-h
Help: You are looking at it.
