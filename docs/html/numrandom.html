NAME

numrandom - Print out a random number.  

SYNOPSIS

numrandom /[expression]/  

DESCRIPTION

numrandom is a program that is part of the numeric utilities package. numrandom will print out a random number determined by the expression that you give. The syntax and program is nearly identical to the range program, except that numrandom picks a number at random from the range expression. If no expression is specified, random will print out a random integer between 1 and 100.  

OPTIONS

-h
Help: You are looking at it.
 

EXAMPLES

Random number from 1 to 10 : $ numrandom /1:10/

From 1 to 10 or from 15 to 20 : $ numrandom /1:10,15:20/

An even number from 0 to 10 : $ numrandom /0:10i2/

An odd number. Notice the starting number in the range : $ numrandom /1:10i2/

A factor of 3 between 99 and 120 : $ numrandom /99:120i3/

A decimal number : $ numrandom /1.1:2.5i0.1/

A negative random number : $ numrandom /0.0:-2.0i0.3/
