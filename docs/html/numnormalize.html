NAME

numnormalize - Normalize a set of numbers. By default between 0 and 1.  

SYNOPSIS

numnormalize [-hR] <FILE>

| numnormalize [-hR] (Input on STDIN from pipeline.)

numnormalize [-hR] (Input on STDIN. Use Ctrl-D to stop.)  

DESCRIPTION

numnormalize is a program that is part of the numeric utilities package. numnormalize will take a set of numbers on input and return that set as a normalized set of numbers between 0 and 1 by default. Or you can use the -R option to specify a different normalized range.  

OPTIONS

-R <range>
This allows you to specify a different normalized range instead of from 0 to 1. For example -R 0..5
-h
Help: You are looking at it.
