num-utils-ng                                                               

Description

The purpose of this project is to analyze and improve the conceptions choices used for the num-utils library development. This library implement severals numbers of numeric algorithms from float data.
An important point of this project is the performances improvement.

The 'num-utils' are a set of programs for dealing with numbers from the Unix command line. Much like the other Unix command line utilities like grep, awk, sort, cut, etc. these utilities work on data from both standard in and data from files.

Programs (click on the link to see its man page in html format)

  • numaverage: A program for calculating the average of numbers.
  • numbound: Finds the boundary numbers (min and max) of input.
  • numinterval: Shows the numeric intervals between each number in a sequence.
  • numgrep: Like normal grep, but for sets of numbers.
  • numnormalize: Normalizes a set of numbers between 0 and 1 by default.
  • numprocess: Do mathmatical operations on numbers.
  • numrandom: Generate a random number from a given expression.
  • numrange: Generate a set of numbers in a range expression.
  • numround: Round each number according to it's value.
  • numsum: Add up all the numbers.
